## 🚨 The Problem

College students have long been stuck in an exhausting loop of paperwork, follow-up emails, and zero transparency when it comes to refunds. Whether it's a fest registration that got cancelled, a hospital visit covered under student insurance, or a mess rebate that never showed up — the process is manual, opaque, and often years behind. There's no single place to raise a claim, track its status, or hold the institution accountable.

---

## ✅ What RefundHub Does

RefundHub is a full-stack web application that centralises and automates the three most common college reimbursement workflows:

### 🎪 Fest Refunds
Submit and track reimbursements for cancelled or unused college fest registrations.

### 🏥 Hospital Insurance Claims
Upload medical bills and raise insurance reimbursement requests covered under student health schemes.

### 🍽️ Mess Rebates
Apply for mess fee rebates for periods of absence, outstation, or leave — with document proof uploads.

## ⚙️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + Tailwind CSS |
| Backend | Node.js + Express.js (REST API) |
| Database | MongoDB (via Mongoose) |
| File Storage | Cloudinary (documents, bills, proof uploads) |

---

## ✨ Key Features

- Role-based access for students and college admins
- Three dedicated claim modules — Fest, Hospital, Mess
- Real-time claim status tracking (Pending → In Review → Approved / Rejected)
- Secure document uploads via Cloudinary
- Admin dashboard to review, approve, and manage all claims
- Clean, responsive UI built with React and Tailwind
